# tech-sales-dashboard

A web-based interactive dashboard designed to track, visualize, and analyze key sales metrics for tech sales teams. This project demonstrates how HTML, CSS and JavaScript used to build an intuitive and dynamic sales tracking tool.

---

## Features

- Lead Tracking: Input leads for client details: Record clients Name, Company, Email and current Status.
- Closed Deals Tracking: Record the number of deals closed.
- Sales revenue calculator:
  - Total leads
  - Conversion rate
  - Average deals size 
  - Estimated revenue
- Responsive Design: Works on desktops and mobile devices.
- User-Friendly Interface: Sales metrics dashboard and sales revenue calculator.

---

## Technologies Used

- HTML – structure of the dashboard.
- CSS3 – styling and layout.
- JavaScript – metrics dashboard, calculations and dynamic content.
